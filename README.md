# spring_boot_demo

## springboot 2.0.4
前端模板使用spring的thymeleaf

使用bootstrap 和 jquery

## redis
使用redis缓存

## GraphQL
使用graphQl提供外部api

findAll查询没有设置权限，可以随便查询
findByUsername设置了权限，查询需要传递token，需要使用postman等工具进行模拟

## Rest
rest api

## shiro + jwt
使用shiro控制权限，jwt来校验和生成token

token缓存在redis中，有效期1天，自动续命
